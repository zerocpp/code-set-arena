# CodeSetArena

CodeSetArena 是面向 LLM 时代程序设计课程的本地双端系统。

## 课程背景

传统程序设计课程通常采用“教师出题、学生提交代码”的形式；但在 LLM 已经能够完成大量常规编程题的背景下，这种训练方式越来越难以帮助学生理解模型能力边界。

CodeSetArena 采用“角色反转”的课程设计：学生不再只是解题者，而是为 LLM 设计编程题、构造测试数据、记录模型运行证据，并通过同伴审稿和修订迭代来发现 LLM 的失败模式（failure modes）。课程目标是让学生在真实模型交互中学习如何提出高质量问题、验证程序行为、分析模型错误，并形成可复查的实验证据。

## 系统功能简介

CodeSetArena 包含学生端和助教端，两端数据相互独立，通过 `.tar.gz` 包交换课程产物。

学生端支持题目编写、样例与测试数据维护、参考答案校验、模型运行、原始提示词和响应证据保存、匿名审稿、审稿反馈阅读、题目修订和最终提交。学生端会保留模型调用的必要 raw 证据，但不会把 `API_KEY` 写入提交包。

助教端支持学生包导入与校验、匿名审稿任务分配、审稿结果收集、修订反馈发放、正式评测和课程统计。助教端可用于组织 Stage 1 出题、Stage 2 审稿、Stage 3 修订和 Stage 4 正式评测的完整课程流程。

当前版本为 `v7.1.7`。本地 Docker 镜像按 Linux 容器架构区分，发布包提供学生端和助教端的 `linux-amd64`、`linux-arm64` 版本。模型服务参数必须在对应端的设置页或 CLI 中手动填写并保存；系统不会从发布包 `.env`、容器环境变量或默认值初始化 `BASE_URL / API_KEY / MODELS`。
